%%%%
flame cloud

一团猛烈燃烧的火云。任何生物，
如果没有对火的完全免疫，身处其中就会受到伤害。

穿过此云的远程寒冰攻击会被降低射程，并同时消散了云层。

如果在水边或水上释放，高温会蒸腾出大量蒸汽。
%%%%
noxious fumes cloud

一团呛人又有毒的云。
任何生物身处其中，都有被气体混乱的危险，除非具有毒抗。
越坚韧、经验等级越高的怪物不太会受此影响。
%%%%
freezing vapour cloud

一团冰气云。任何生物，如果没有对寒的完全免疫，身处其中就会受到伤害。

穿过此云的远程火焰攻击会被降低射程，并同时消散了云层。
%%%%
poison gas cloud

一个有着致命毒性的云。任何没有毒抗的生物，在里面就会中毒并直接受到伤害。

%%%%
translocational energy cloud

这是某个最近来过这里的生物的残留物。
但它已经不在了，被放逐到附近，或穿过楼层，或者脱离现实，直到深渊。
一些巫师说，这是宇宙产生的一种“过敏反应”，
是对魔法以不可能的方式扭曲空间的反应；一个宇宙的喷嚏。
也许在形而上学意义上是恶心的，但在其他方面无害。
%%%%
spreading flames cloud

一团熊熊的森林大火。任何生物，如果没有对火的完全免疫，
身处其中就会受到伤害。火焰会间歇地爆发，并让周围的环境充满更小的火。
不幸的是，附近的树木，常常自己会变成更多的火种。

穿过此云的远程寒冰攻击会被降低射程，并同时消散了云层。

如果在水边或水上释放，高温会蒸腾出大量蒸汽。
%%%%
steam cloud

一团灼热的蒸汽云。在其中的生物会轻微受伤，火抗属性会大大抵消伤害。
%%%%
ink cloud

一团墨云，里面完全是水。
%%%%
calcifying dust cloud

一个石化云。在云中呆任何时长的生物都会发现自己在变成石头，
尽管更有经验的生物也许能够抵抗这种影响。
%%%%
blessed fire cloud

一团圣火云，充满了神圣的力量。那些服侍善神的将不受影响。
其他生物会受伤，而那些亡灵和恶魔将受到很大的伤害。

rain cloud

一个极度集中的暴风雨。它可能形成浅水池，或临时加深现有水池。
完全由火形成的生物不愿在雨中呆太久。
%%%%
mutagenic fog cloud

一团纯净、无结构的魔法，凝结成一团闪烁的薄雾，本质上是有毒的魔法废料。
冒险进入其中的生物会发现自己逐渐受到魔法辐射的污染，
当消散时，足够多的辐射会造成负面变异。
%%%%
magical condensation cloud

某种高度集中的魔法能量被唤醒，本身是完全无害的。
%%%%

只有一部分伤害能通过寒抗而减轻。
%%%%
